### 术语表
| 术语 | 定义 |
|---|---|
| 双边市场 | 同时服务用户（观众）与主播（内容生产者）的平台 |
| 高潜主播 | 长期LTV或生存概率高于阈值的主播 |
| 启动期 | 开播后前10-30分钟，决定该场流量分配的关键窗口 |
| 三段式机制 | 播种(Seed)→验证(Validate)→加码(Scale) |
| 边际收益 | 给额外流量后带来的增量长期收益 |
| Bandit | 探索-利用算法（Contextual Bandit） |
| 成长信号 | 非金钱激励：粉丝增长、互动反馈、里程碑等 |

### 背景
- US直播市场特性：打赏收入有限，主播动机需靠成长/认同/粉丝反馈支撑
- 核心约束：不能为高付费场景牺牲低付费/纯观看用户体验
- 核心目标：在约束下实现用户体验 + 主播留存的双赢
